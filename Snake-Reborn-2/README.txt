        Commands:

 enter -> respawn
 esc -> back to menu
-------------------------------
 up arrow    -> go up
 down arrow  -> go down
 right arrow -> go right
 left arrow  -> go left
-------------------------------
 W -> go up
 A -> go left
 S -> go down
 D -> go right
 E -> rotate right
 Q -> rotate left
-------------------------------

Minimum requirements:

 - Operative System at 32 or 64 bit
 - Java 8 or later
 - 800 MHz CPU
 - 250 MB of free RAM
 - 30 MB of free storage
